# Whack-a-Mole Game

## Overview

Whack-a-Mole Game is a browser-based arcade game developed using HTML, CSS, and JavaScript. Players earn points by clicking moles that randomly appear on the screen before they disappear.

---

## Features

- Random mole generation
- Score tracking
- Interactive gameplay
- Responsive design
- Simple UI

---

## Technologies Used

- HTML
- CSS
- JavaScript

Open index.html in your browser.
